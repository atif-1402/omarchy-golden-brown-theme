# Golden Brown

A warm muted sepia theme with deep earthy tones and soft golden highlights, inspired by autumn leaves, vintage terminals, and dark bronze aesthetics.

[charcoal.nvim](https://github.com/mubin-thinks) by [Mubin-thinks](https://github.com/mubin-thinks)

To install this theme run this command 

```bash
omarchy-theme-install https://github.com/atif-1402/omarchy-golden-brown-theme.git
```
